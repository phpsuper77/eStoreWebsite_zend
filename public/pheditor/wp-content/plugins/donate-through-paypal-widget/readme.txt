== Description ==

A wordpress plugins to add paypal donation widget in the sidebar . It has been developed for not profiting organizations. Very simple to use . You can also add certain text before your donation banner via plugin options

== Installation ==

1. Upload `plugin` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1.go to settings -> donate through paypal widget and add your options .
